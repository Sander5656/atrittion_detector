HR Deserción — Aplicación de Regresión Logística
Aplicación de escritorio en CustomTkinter que entrena y evalúa un pipeline de Regresión Logística sobre un dataset de deserción de empleados. Incluye preprocesado automático, búsqueda de hiperparámetros, visualizaciones (matriz de confusión, ROC, Precision‑Recall, métricas) y una interfaz para ingresar un registro y obtener una predicción con probabilidad.

Instalación y Dependencias
Requisitos
- Python 3.8 o superior
- Sistema operativo con soporte GUI (Windows, macOS, Linux con X11)
